# 更新日志

## [2.2.0] - July 2026

**摘要：** 与其他应用的系统剪贴板集成、框选、联动分屏、链接处理、受密码保护的 ZIP 压缩包、Windows 原生属性、扩展 API 增强、希伯来语支持，以及导航器体验优化。

### 新功能

#### 系统剪贴板集成

通过系统剪贴板，在 Sigma File Manager 与其他应用之间复制和粘贴文件、文件夹和图像。

- **跨应用文件传输**：在 SFM 中复制或剪切项目，并粘贴到文件资源管理器等应用；也可以使用 `Ctrl+V` 将其他应用中复制的路径和文件粘贴到导航器；
- **粘贴图像**：将浏览器和其他应用中复制的图像直接粘贴到文件夹；
- **冲突对话框**：粘贴目标已存在时，可选择 `重命名` 或 `合并`；对于单个文件冲突，还可选择 `替换`、`跳过`、`两者都保留` 或 `全部应用`；
- **剪贴板工具栏**：可选择在工具栏中预览从其他应用复制的图像和文件路径；

可在 `设置 > 界面外观 > 剪贴板` 中控制工具栏可见性。隐藏工具栏时，仍可用 `Ctrl+V` 粘贴。

![system-clipboard](./public/changelog/assets/2.2.0/system-clipboard.webp)

#### 框选

在导航器空白处拖动，用选择框选中多个项目。

- **修饰键**：按住 `Ctrl` 或 `Shift` 添加到当前选择；按住 `Alt` 反选；
- **更易拖动**：可增大列表内边距和网格间距，留出更多空白区域以便开始拖动；

在 `设置 > 通用 > 文件视图 > 启用框选` 中启用。

![box-selection](./public/changelog/assets/2.2.0/box-selection.webp)

#### 联动分屏

新增 `联动` 分屏模式，提供更简洁的分栏操作方式：在第一个窗格中点击文件夹后，其内容会显示在第二个窗格中。

现有的独立 `分屏` 模式保持不变。可在导航器选项菜单的 `分屏模式` 中选择，或使用 `Ctrl+S` 切换分屏。

信息面板图标也已更新，更易与分屏图标区分。

![linked-split-view](./public/changelog/assets/2.2.0/linked-split-view.webp)

#### 链接处理

从导航器创建并检查文件系统链接。

- **创建链接**：通过上下文菜单（`创建链接`）创建符号链接、快捷方式、硬链接和联接；
- **链接列**：可选的列表列，包括类型、链接、链接目标和链接状态（`有效`、`损坏`、`未知`、`不支持`）；
- **打开行为**：目录快捷方式和符号链接文件夹会导航到其目标；其他链接目标用默认应用打开；

![link-handling](./public/changelog/assets/2.2.0/link-handling.webp)

#### 原生属性窗口

在 Windows 上，可通过上下文菜单、操作菜单、`Alt+Enter` 或 `Alt` + 双击，为所选项目打开系统原生属性对话框。

![native-properties](./public/changelog/assets/2.2.0/native-properties.webp)

#### 列表视图列调整大小与重排

列表视图列可根据使用习惯调整大小和顺序。

- **调整大小**：拖动列边缘更改宽度；
- **重排与可见性**：通过列表标题的 `列` 弹出框管理顺序和可见性；
- **宽度选项**：`填充可用宽度` 和 `设置最小宽度`；

![list-column-resize](./public/changelog/assets/2.2.0/list-column-resize.webp)

#### 根级 `位置` 入口

根级 `位置` 入口会列出驱动器和虚拟位置，方便快速切换。

- **地址栏**：从驱动器根目录向上一级，或通过地址栏或地址编辑器打开 `位置`；
- **收藏夹和标签**：位置可像其他目录一样添加到收藏夹并打标签；
- **分屏**：特别适合在不离开导航器的情况下在窗格间切换驱动器；

![root-locations-address](./public/changelog/assets/2.2.0/root-locations-address.webp)

### 扩展

#### 扩展 API 与视图

扩展现在可以使用更多宿主功能和 UI 组件。

- **本地二进制文件**：通过自动配置或手动选择本地二进制文件来设置扩展依赖（`扩展 > 依赖项`）；
- **HTTP 请求**：扩展可向其清单允许的主机发起 HTTP 请求；
- **视图控制**：获得视图权限后，扩展可应用导航器布局和排序偏好；
- **剪贴板 API**：扩展可读写剪贴板（需权限）；
- **列表-详情视图**：新增由可搜索列表和详情窗格组成的扩展 UI 模式；

### 新设置

- **启用框选**：在空白处拖动以多选；
  `设置 > 通用 > 文件视图 > 启用框选`

  `设置 > 通用 > 文件视图 > 增大文件视图间距`

  `设置 > 通用 > 性能 > 在内存中保留打印窗口`
- **外部图像的剪贴板工具栏**：显示其他应用中复制的图像的剪贴板工具栏；
  `设置 > 界面外观 > 剪贴板`
- **外部路径的剪贴板工具栏**：显示其他应用中复制的文件路径的剪贴板工具栏；
  `设置 > 界面外观 > 剪贴板`
- **动态信息面板大小**：让信息面板大小自动适应内容；手动调整大小后会关闭此功能；
  `设置 > 界面外观 > 信息面板 > 动态信息面板大小`
- **在信息面板预览中显示全尺寸图像**：在信息面板中显示全分辨率图像；
  `设置 > 界面外观 > 信息面板 > 在信息面板预览区域显示全尺寸图片`
- **默认静音视频预览**：浏览时将信息面板视频预览静音；
  `设置 > 界面外观 > 信息面板 > 默认静音视频预览`
- **自动播放视频预览**：选中时在信息面板中自动播放视频；
  `设置 > 界面外观 > 信息面板 > 自动播放视频预览`

#### 归档解压

ZIP 解压现已支持加密压缩包和使用非 UTF-8 编码的文件名。

- **受密码保护的 ZIP**：解压时可根据提示输入压缩包密码；
- **文件名编码**：可在 `压缩包解压选项` 中选择编码。系统会优先自动检测，也可从按地区分组的编码中手动选择；

![archive-extraction-options](./public/changelog/assets/2.2.0/archive-extraction-options.webp)

![archive-extraction-encoding](./public/changelog/assets/2.2.0/archive-extraction-encoding.webp)

#### 网格排序

导航器选项菜单中新增了专用于网格布局的排序选项。

- **排序依据**：名称、项目、大小、修改时间、创建时间、标签、类型、链接和链接状态；
- **方向**：升序或降序，与列表视图排序分开存储；

![grid-sorting](./public/changelog/assets/2.2.0/grid-sorting.webp)

#### Shell 扩展

上下文菜单现在可在 `Shell 扩展` 下显示其他应用注册的新式 Shell 扩展操作。

#### 导航器性能

浏览大型文件夹和媒体时更快，占用内存更少。

- **首次加载**：打开目录时的首次加载速度更快；
- **图标加载**：自定义和系统图标出现延迟更小；
- **列表滚动**：大型目录中的列表滚动更流畅；
- **媒体预览**：图像、GIF 和视频预览响应更快，内存占用更少；
- **索引**：全局搜索索引更稳定；

#### 首页与上下文菜单

- **断开连接**：在支持时从上下文菜单断开网络或可移动挂载；

- **右键取消选择**：右键点击导航器空白背景会在打开背景菜单前取消当前选择；
- **首页操作**：点击操作后，首页上下文菜单会自动关闭；选择 `在新标签页中打开` 会打开导航器，并自动将新标签页滚动到可视区域；
- **窗口拖动区域**：在 Linux 风格标题栏上，拖动区域延伸到工具栏按钮，便于移动窗口；

## [2.1.0] - May 2026

**摘要：** 导航器性能提升、生成缩略图、扩展提供的主题、打印、文件预览、新快捷键、地址编辑器改进、状态中心重新设计，以及标签页和导航体验优化。

### 新功能

#### 打印

通过上下文菜单、操作菜单或 `Ctrl+O`，直接从导航器打印所选文件。

- **支持的格式**：图像、PDF、文本格式；
- **快速退出**：使用 `Escape` 关闭打印视图；

![printing](./public/changelog/assets/2.1.0/printing.webp)

#### 拖放到标签页

将文件或文件夹拖到标签页上，以移动或复制到另一个标签页的目录。

- **目标标签页**：在导航器中拖动文件时，可将标签页作为放置目标；
- **悬停激活**：拖动时悬停在标签页上可在放置前切换到该标签页；
- **分屏标签页**：目录标签页组可继续作为放置目标，同时保留原有的分屏标签页结构；

![file-drop-to-tabs](./public/changelog/assets/2.1.0/file-drop-to-tabs.webp)

#### 信息面板中的文件预览

信息面板现在可以预览所有 Quick View 支持的文件类型，而不仅是图像和视频。

- **媒体预览**：图像使用生成的缩略图，视频和音频包含原生控件，PDF 内联渲染；
- **文本预览**：在安全的大小限制内，对文本文件解码并显示紧凑预览；
- **不支持的内容**：不支持的文件和文件夹仍显示简单的图标占位符；

![info-panel-file-preview](./public/changelog/assets/2.1.0/info-panel-file-preview.webp)

#### 导航器列表视图列

列表视图新增了更多可选列，也可以直接在列表中更方便地管理元数据。

- **创建时间列**：显示并按创建日期排序；
- **标签列**：直接在列表视图中显示标签，并从该列添加、移除或编辑标签；

![navigator-list-columns](./public/changelog/assets/2.1.0/navigator-list-columns.webp)

### 扩展

#### 来自扩展的应用主题

扩展现在可以提供完整的应用配色主题。已安装的主题扩展会显示在主题选择器中。

#### 来自扩展的图标主题

扩展现在可以为文件夹和文件提供导航器图标主题。

- **分别选择**：在 `设置 > 界面外观 > 图标主题` 中独立选择文件夹和文件图标主题；
- **内置与扩展主题**：可使用内置的默认或系统图标主题，也可使用已启用扩展提供的主题；
- **主题匹配**：扩展提供的主题可按文件扩展名、文件名、文件夹名和文件夹展开状态定义图标；

### 新设置

- **加粗当前标签页文字**：将当前标签页标题设为粗体（`设置 > 标签页 > 标签外观 > 加粗当前标签页文字`）；

### 用户体验改进

#### 大型目录性能

导航和快速搜索响应更快，浏览包含大量媒体文件的文件夹时内存占用也更低。

- **生成缩略图**：为图像和视频生成较小尺寸的缩略图，无需在每个文件卡片中加载完整媒体；
- **渐进式图像**：网格图像卡片可在最终缩略图就绪前显示模糊的低分辨率缩略图；
- **取消缩略图生成**：文件夹或可见条目发生变化时，取消不再需要的缩略图生成任务；
- **渲染性能**：大型目录条目使用更高效的渲染，Quick View 使用生成缩略图和虚拟列表；

![low-res-image-thumbnail-preview](./public/changelog/assets/2.1.0/low-res-image-thumbnail-preview.webp)

#### 快速搜索

快速搜索现在提供两种模式：被动模式和主动模式：

- **被动模式**：开始键入时自动激活。无需将焦点移到搜索框即可筛选条目，也不会妨碍导航操作。
- **主动模式**：使用 `Ctrl+F` 激活。焦点会移到搜索框并暂停导航操作，以便更精细地编辑查询内容。

其他更改：

- **键入以筛选**：键入字母数字键现在始终在活动窗格中启动快速搜索（被动模式）；
- **键盘导航**：自动选中第一个匹配项；
- **弹出设计**：快速搜索弹出框更紧凑，并避免遮挡目录项目；

![quick-search](./public/changelog/assets/2.1.0/quick-search.webp)

#### 地址编辑器

地址编辑器现在可以更灵活地打开各种路径。

- **文件和目录**：从地址编辑器打开文件以及目录；
- **常用路径**：切换到专门用于快速打开常用路径的模式；
- **建议**：浏览目录条目、精确匹配、最近路径、已标记项目、用户文件夹和系统驱动器；
- **键盘操作**：从编辑器向后、向前、向上导航，并在父目录中显示条目；

![address-editor](./public/changelog/assets/2.1.0/address-editor.webp)

#### 状态中心

您现在可以将 SFM 设置为 Windows 上的默认文件管理器（`设置 > 实验性功能`）。启用此设置后，大多数系统文件操作将路由到 SFM：

- 文件资源管理器应用图标；
- `Ctrl+E` 快捷键；
- 在文件夹中显示文件；

#### 局域网共享

直接从应用在本地网络上共享和串流文件及目录。

从导航器中的工具栏按钮或从任何文件或目录的上下文菜单访问局域网共享。共享激活后，将显示二维码和可共享的 URL。有两种模式可用：

- **串流**：通过网页浏览器将文件和目录串流到网络上的任何设备；
- **FTP**：通过 FTP 共享文件，以便从其他应用直接访问。您可以从其他设备向计算机下载和上传文件；

#### 快速访问菜单

侧边栏中的"仪表板"按钮现在充当快速访问菜单。将鼠标悬停在上面会打开一个显示您的收藏夹和已标记项目的面板。

面板中的所有项目都是真实的目录条目——您可以拖放项目进出、右键打开上下文菜单，并执行任何标准文件操作。

可以在 `设置 > 界面外观 > 悬停时打开快速访问面板` 中禁用。

#### Zip 压缩包

直接从文件浏览器操作菜单压缩和解压 zip 压缩包：

- **解压**：将 `.zip` 文件解压到当前目录或命名文件夹；
- **压缩**：将选中的文件和目录压缩为 `.zip` 压缩包；

#### WSL 驱动器检测

在 Windows 上，应用现在会自动检测已安装的 WSL 发行版并在导航器中显示其驱动器，让您可以原生浏览 WSL 文件系统。

#### 标签编辑

您现在可以编辑标签名称和颜色。在任何文件或目录上打开标签选择器，即可重命名标签、更改颜色或删除标签。

#### 应用内更新

您现在可以直接从更新通知中下载和安装更新，无需离开应用。

#### 复制路径

在文件和目录上下文菜单中添加了"复制路径"选项。
